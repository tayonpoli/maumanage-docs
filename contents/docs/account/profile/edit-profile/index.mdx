This section provides a clear, structured **step-by-step guide** for editing user profile within your MauManage application.

## Steps

To edit an user profile, follow these steps:

<Step>
  <StepItem title="Navigate to Profile Menu">
    Go to the **Profile menu** within the MauManage application.
  </StepItem>

  <StepItem title="Edit Profile">
    Click on "**Edit**" button in the top-right of personal information card.
  </StepItem>

  <StepItem title="Modify Fields">
    Change the necessary **fields** in the edit profile form.
  </StepItem>

  <StepItem title="Update Profile">
    Click "**Update**" to submit the form and save the changes to the profile.
  </StepItem>
</Step>
