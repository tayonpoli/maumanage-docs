This section provides a clear, structured **step-by-step guide** for viewing user profile and company profile information within your MauManage application.

## Steps

To view user profile, follow these steps:

<Step>
  <StepItem title="Navigate to Profile Menu">
    Click the "account" button in the user navigation at the bottom of sidebar.
  </StepItem>

  <StepItem title="View User Profile">
    You will see the **personal information** displayed.
  </StepItem>

  <StepItem title="View Company Profile">
    You can also view the **Company information** displayed.
    <Note title="Information" type="danger">
        Only the user with **Owner** role can view the company information and also edit the company information.
    </Note>
  </StepItem>
</Step>
