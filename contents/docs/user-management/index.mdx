    </Note>

## Steps

This section provides a clear, structured **step-by-step guide** for viewing registered users and invitation sent.

<Tabs defaultValue="js" className="pt-5 pb-1">
  <TabsList className="">
    <TabsTrigger value="js">User</TabsTrigger>
    <TabsTrigger value="ts">Invitation</TabsTrigger>
  </TabsList>
  <TabsContent value="js">
To view user management, follow these steps:

<Step>
  <StepItem title="Navigate to User Management Menu">
    Go to the **User Management menu** within the MauManage application.
  </StepItem>

  <StepItem title="View Registered Users">
    You will see a **list of all registered users** in the system.
  </StepItem>

  <StepItem title="View User Details">
    To see more details about a specific user, click on their entry in the list.
  </StepItem>

    <StepItem title="Delete User Invitation">
    Locate the user you wish to delete and click on "**Delete**" in its corresponding row.
  </StepItem>
</Step>
  </TabsContent>
  <TabsContent value="ts">
  To view list of user invitation, follow these steps:

<Step>
  <StepItem title="Navigate to User Management Menu">
    Go to the **User Management menu** within the MauManage application.
  </StepItem>

  <StepItem title="View User Invitation">
    Select the "invitation" tabs and the list of user invitation will display.
  </StepItem>

  <StepItem title="View Invitation Details">
    To see more details about a specific invitation, click on their entry in the list.
  </StepItem>

  <StepItem title="Delete User Invitation">
    Locate the invitation you wish to delete and click on "**Delete**" in its corresponding row.
  </StepItem>
</Step>
  </TabsContent>
</Tabs>
